MedMemo

The Physician-Nurse Hospital Communication System, also known as MedMemo, is a web application designed to bridge the communication gap between physicians and nurses. Often, nurses have a difficult time contacting the physician, which results in wasted time and effort. Even when the physician receives a notification, they either will not be able to respond in a timely manner, or will not know which nurse called for them at all. This application solves these problems while providing a user friendly interface, a seamless communication network, and strong security measures.
